## Rodando o site localmente

> Necessário possuir Docker instalado.

Execute o comando:

`docker-compose up`

Acesse:

[http://localhost:4000](http://localhost:4000)

> Apenas os últimos 15 artigos são exibidos devido ao longo tempo de compilação
